# RNA-seq Analysis Pipeline

## Overview

This repository contains a reproducible RNA-seq workflow for:

* Quality Control
* Alignment
* Read Quantification
* Differential Expression Analysis
* Visualization

## Workflow

FASTQ

↓

FastQC

↓

STAR Alignment

↓

FeatureCounts

↓

DESeq2

↓

Visualization

↓

Report Generation

## Tools

* FastQC
* MultiQC
* STAR
